## Testing network

This section contains key information about the testing network which can be used to test our product.

### Prerequisites

- Generate and have an Ethereum private key
- Option: Have a wallet you can use in your web browser

### Get testnet ETH

Instructions about how to get testnet ETH can be found here: https://docs.scroll.io/en/developers/faucet

For a no-registration method, we recommand using the POW faucet with a wallet like METAMASK to get testnet ETH on Ethereum Sepolia (https://sepolia-faucet.pk910.de/) then bridge these test ETH to Scroll Sepolia using the official bridge (https://portal-sepolia.scroll.io/bridge) and finally send these test ETH from your METAMASK wallet to the private key you use to test.
